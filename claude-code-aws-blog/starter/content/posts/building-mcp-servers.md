# Building Custom MCP Servers for Claude Code

Learn how to build Model Context Protocol servers that extend Claude Code's capabilities.

## What is MCP?

Model Context Protocol (MCP) is an open standard that allows AI applications to connect to external data sources and tools.

## Creating Your First Server

MCP servers can be built in any language to add custom functionality to Claude Code.
